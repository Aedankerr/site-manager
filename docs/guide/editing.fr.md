# Modification du contenu

## Profil / En-tete

Cliquez sur l'**icone de modification** dans la section d'en-tete. Vous pouvez definir :

- **Nom** et **Initiales** (affiches dans le cercle d'avatar si aucune photo n'est telechargee)
- **Titre** (votre role/poste principal)
- **Sous-titre** (contexte supplementaire, par ex. departement ou specialisation)
- **Biographie** (resume professionnel — les sauts de ligne sont preserves)
- **Lieu**
- **E-mail**, **Telephone**, **LinkedIn** (affiches sous forme de badges de contact)
- **Langues** (langues parlees, par ex. « Anglais, Francais, Allemand » — affiche sous forme de badge avec une icone de globe)

## Photo de profil

Dans la boite de dialogue de modification du profil, vous pouvez telecharger une photo de profil (JPEG, PNG ou WebP). L'image remplace le cercle d'initiales. Pour la supprimer, utilisez l'option de suppression dans la meme boite de dialogue.

## Experience professionnelle

Chaque entree d'experience comprend :

- **Intitule de poste** et **Nom de l'entreprise**
- **Date de debut** et **Date de fin** (laissez la date de fin vide pour les postes actuels — affiche « Present »)
- **Lieu** et **Code pays** (code a 2 lettres comme `us`, `ch`, `fr` — utilise pour les icones de drapeaux sur la chronologie)
- **Points cles** — puces decrivant vos realisations (une par ligne)

## Logos d'entreprise

Chaque entree d'experience peut avoir un **logo d'entreprise**. Les logos apparaissent sur la chronologie sous forme d'images au lieu de simples noms d'entreprise.

### Telecharger un logo

1. Modifiez une entree d'experience
2. Dans la section **Logo d'entreprise**, cliquez sur **Choisir une image** pour telecharger un fichier (JPEG, PNG ou WebP, 5 Mo max)
3. Une petite image carree fonctionne le mieux

### Reutiliser des logos

Cliquez sur **Utiliser un existant** pour ouvrir la grille de selection de logos, qui affiche tous les logos precedemment telecharges. Cliquez sur l'un d'eux pour le reutiliser — cela evite de telecharger le meme logo plusieurs fois.

### Remplissage automatique

Lorsque vous saisissez un nom d'entreprise qui possede deja un logo (provenant d'une experience actuelle ou enregistree), le logo est automatiquement renseigne. Cela fonctionne entre les jeux de donnees.

### Propagation globale

Lorsqu'un logo est defini, un bouton bascule apparait : **« Synchroniser le logo sur toutes les experiences [Entreprise] »**. Lorsqu'il est active :

- Le logo est applique a **toutes** les experiences portant le meme nom d'entreprise — y compris celles des jeux de donnees enregistres
- Les futures experiences pour la meme entreprise heriteront automatiquement du logo
- Le desactiver ne **supprime pas** les logos deja appliques — cela arrete uniquement la propagation automatique future

### Supprimer un logo

Cliquez sur le bouton **Supprimer** dans la section logo pour dissocier un logo d'une experience. Le fichier est conserve sur le disque pour reutilisation. Pour supprimer definitivement un fichier de logo inutilise, ouvrez le selecteur de logos et cliquez sur le bouton de suppression sur tout logo qui n'est pas utilise.

## Certifications

Chaque certification a un **nom**, un **organisme**, une **date de delivrance**, et optionnellement une **date d'expiration** et un **identifiant de certification**.

## Formation

Chaque entree a un **diplome/titre**, un **etablissement**, des **dates de debut/fin**, et une **description** optionnelle.

## Competences

Les competences sont organisees en **categories** (par ex. « Langages de programmation », « Outils et plateformes »). Chaque categorie contient des etiquettes de competences individuelles. Les categories peuvent avoir une **icone** selectionnee dans une liste predeterminee : code, serveur, base de donnees, cloud, parametres, utilisateurs, mallette, processeur, couches, ou par defaut. Si aucune icone n'est choisie, le systeme en detecte automatiquement une en fonction du nom de la categorie (par ex. une categorie nommee « Programmation » obtient automatiquement l'icone code).

## Projets

Chaque projet a un **titre**, une **description**, des **technologies** (affichees sous forme d'etiquettes) et un **lien externe** optionnel.

## Visibilite des elements

Chaque element individuel (experience, certification, entree de formation, etc.) dispose d'un **bouton de visibilite**. Les elements masques sont exclus de l'impression/PDF et de la vue publique, mais restent dans l'administration pour une utilisation future.

## Reordonnancement par glisser-deposer

Dans la plupart des sections, les elements peuvent etre reordonnes en les faisant glisser. L'ordre de tri est enregistre automatiquement.
