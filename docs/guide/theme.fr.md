# Theme et apparence

## Couleur du theme

Cliquez sur le bouton **Theme** dans la barre d'outils pour ouvrir le selecteur de couleur. Vous pouvez :

- Choisir une couleur dans la **roue chromatique**
- Ajuster la **luminosite** avec le curseur
- Entrer un **code hexadecimal** specifique
- Choisir parmi **8 couleurs predefinies** (Bleu, Emeraude, Violet, Rouge, Orange, Cyan, Indigo, Rose)

La couleur du theme affecte les titres, bordures, liens, etiquettes et elements d'accentuation dans l'ensemble du CV.

## Format de date

Allez dans **Parametres → Avance → Format de date** pour choisir l'affichage des dates sur votre CV :

| Format | Exemple |
|--------|---------|
| MMM YYYY | Jan 2020 |
| MMM YY | Jan 20 |
| MMMM YYYY | January 2020 |
| MM/YYYY | 01/2020 |
| MM.YYYY | 01.2020 |
| MM-YYYY | 01-2020 |
| YYYY-MM | 2020-01 |
| YYYY | 2020 (annee uniquement) |
