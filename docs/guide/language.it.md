# Lingua

Fare clic sull'**icona del globo** nella barra degli strumenti per cambiare la lingua dell'interfaccia admin. Un menu a griglia mostra tutte le lingue disponibili — fare clic su una per applicarla immediatamente.

**Lingue supportate:** Inglese, Tedesco (Deutsch), Francese (Francais), Olandese (Nederlands), Spagnolo (Espanol), Italiano (Italiano), Portoghese (Portugues), Cinese (中文).

L'impostazione della lingua influisce solo sull'interfaccia admin — non modifica il contenuto del CV. La preferenza viene salvata e mantenuta tra le sessioni.
