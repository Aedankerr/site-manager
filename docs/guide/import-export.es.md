# Importación y exportación

## Exportar su CV

Haga clic en **Export** en la barra de herramientas para descargar su CV completo como un archivo JSON. Esto incluye todas las secciones, elementos, configuraciones y secciones personalizadas. Utilícelo para copias de seguridad o para transferir su CV a otra instancia.

## Importar datos

Haga clic en **Import** y seleccione un archivo JSON exportado previamente. Esto reemplaza los datos actuales de su CV con los datos importados. Las secciones personalizadas y todas las configuraciones están incluidas.

!!! tip
    Exporte su CV antes de importar, para tener una copia de seguridad del estado actual.
