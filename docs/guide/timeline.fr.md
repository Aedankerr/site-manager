# Chronologie

La chronologie est generee automatiquement a partir de vos experiences professionnelles. Elle affiche :

- Les logos d'entreprise (ou les noms d'entreprise si aucun logo n'est defini)
- Les intitules de poste
- Les periodes
- Les drapeaux de pays (lorsque le pays change entre les experiences)

## Navigation par clic sur la chronologie

Cliquer sur un element de la chronologie fait defiler la page jusqu'a la carte d'experience correspondante et la met brievement en surbrillance.

## Emplois simultanes (embranchement de la chronologie)

Lorsque deux ou plusieurs experiences professionnelles se chevauchent dans le temps d'au moins 1 mois, la chronologie les affiche automatiquement sous forme de **pistes paralleles** :

- Le poste concurrent se separe sur une **piste secondaire** — une ligne sureleve au-dessus de la chronologie principale
- Des **connecteurs en courbe en S** montrent visuellement ou la branche se separe et rejoint la piste principale
- Si un poste parallele est en cours (sans date de fin), la ligne de branche s'etend jusqu'au bord droit de la chronologie
- Les cartes des elements sur la piste secondaire sont positionnees separement pour eviter le chevauchement avec les cartes de la piste principale

Aucune configuration n'est necessaire — l'embranchement est entierement automatique en fonction des dates de vos experiences. Les chevauchements courts de moins d'1 mois (frequents lors des transitions professionnelles) sont intentionnellement ignores.

## Format de date de la chronologie

Par defaut, la chronologie affiche uniquement les annees (par ex. « 2020 - 2023 »). Vous pouvez modifier cela dans **Parametres → Avance → Chronologie : annees uniquement**. Lorsque cette option est desactivee, la chronologie utilise le meme format de date que le reste de votre CV.
