# Compatibilidade ATS

O CV Manager gera saída compatível com ATS (Applicant Tracking System):

- **Marcação Schema.org** — dados estruturados que sistemas ATS podem analisar (Person, OrganizationRole, EducationalOccupationalCredential, etc.)
- **HTML semântico** — hierarquia adequada de títulos, elementos article e listas
- **Bloco ATS oculto** — uma versão em texto simples do seu CV é incorporada na página para analisadores que não processam HTML estilizado
- **Saída de impressão limpa** — sem poluição visual, hierarquia de conteúdo adequada

Nenhuma configuração especial é necessária — a otimização ATS está sempre ativa.
