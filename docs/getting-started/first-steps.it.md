# Primi passi

Dopo aver installato CV Manager, seguire questi passaggi per configurare il proprio CV:

1. Aprire l'interfaccia admin all'indirizzo `http://your-server:3000`
2. Fare clic sull'**icona di modifica** nell'intestazione per impostare nome, titolo, biografia e informazioni di contatto
3. Aggiungere le proprie esperienze lavorative, formazione, certificazioni, competenze e progetti
4. Utilizzare il pulsante **Tema** nella barra degli strumenti per scegliere un colore adatto al proprio stile
5. Premere **Stampa / PDF** per esportare, oppure condividere l'URL pubblico sulla porta 3001

!!! tip "Suggerimento"
    Il sito pubblico sulla porta 3001 serve una versione di sola lettura del CV. Condividere quell'URL con recruiter e datori di lavoro — non vedranno alcun controllo di amministrazione.
