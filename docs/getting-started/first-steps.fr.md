# Premiers pas

Apres avoir installe CV Manager, suivez ces etapes pour configurer votre CV :

1. Ouvrez l'interface d'administration a l'adresse `http://votre-serveur:3000`
2. Cliquez sur l'**icone de modification** dans l'en-tete pour definir votre nom, titre, biographie et coordonnees
3. Ajoutez vos experiences professionnelles, formations, certifications, competences et projets
4. Utilisez le bouton **Theme** dans la barre d'outils pour choisir une couleur qui correspond a votre style
5. Cliquez sur **Imprimer / PDF** pour exporter, ou partagez l'URL publique sur le port 3001

!!! tip "Conseil"
    Le site public sur le port 3001 affiche une version en lecture seule de votre CV. Partagez cette URL avec les recruteurs et les employeurs — ils ne verront aucun controle d'administration.
